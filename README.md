# Personal Academic Homepage - Dr. Hong-Yuan Guo

This repository contains the source code for the personal academic homepage of Dr. Hong-Yuan Guo, Research Assistant Professor at The Hong Kong Polytechnic University (PolyU).

## Features
- **Modern Design**: High-impact Klein Blue light theme.
- **Responsive**: Optimised for both desktop and mobile viewing.
- **Rich Content**: Includes sections for:
    - Academic Background & Work Experience
    - Research Grants (PI/Co-PI projects)
    - Full Publications List (Categorized)
    - Honors & Awards

## Technology Stack
- Clean HTML5 & Vanilla CSS3.
- Scroll Reveal animations for professional UX.

## Deployment
This site is intended to be hosted via **GitHub Pages**.
